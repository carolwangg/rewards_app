# Contributor Setup
## Dependencies
- Node.js
- npm
- git
- Visual Studio Code (VSCode)

## Steps
1. Clone the repo
```git clone https://github.com/carolwangg/rewards_app.git```
2. Open the repo in VSCode
3. Install dependencies by opening terminal and running
```cd RewardsApp```
```npm install```
4. Install Expo Go app onto mobile device from the app store
5. Run the rewards app through Expo Go
```npx expo start```
6. Scan the QR code generated and open app on mobile device. The QR code should redirect you to the Expo Go app.
> [!IMPORTANT]  
> App has been tested on iOS but not on Android yet
